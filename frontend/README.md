# Frontend / SCADA-UI\n\nDenna komponent är det grafiska användargränssnittet (GUI) för systemet.\n\n## Funktioner\n- Dynamiska processvyer (SCADA) som visar realtidsdata.\n- Interaktiva grafer för historisk dataanalys.\n- Larmvy för att visa och hantera aktiva och historiska larm.\n- Gränssnitt för att konfigurera systemet (PLC-enheter, taggar, användare etc.).\n- Responsiv design för att fungera på både datorer och mobila enheter.
