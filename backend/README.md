# Backend / Kommunikation\n\nDenna komponent ansvarar för all kommunikation med PLC-enheter.\n\n## Protokoll\n- Mitsubishi FX5U (via SLMP)\n- Modbus TCP\n\n## Ansvar\n- Etablera och upprätthålla anslutningar till konfigurerade PLC:er.\n- Pollning av definierade PLC-register och adresser.\n- Formatera och vidarebefordra insamlad data till databas-lagret.\n- Hantera anslutningsfel och återanslutningslogik.
