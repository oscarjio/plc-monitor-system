# PLC Monitoring System\n\nDetta projekt är ett modernt, enkelt och administratörsvänligt system för monitorering av PLC-enheter.\n\n## Mål\n- Centraliserad insamling av data från diverse PLC-enheter.\n- Realtidsvisualisering av processdata.\n- Historisk datalagring för analys och grafer.\n- Larmhantering.\n- AI-baserad felsökning och proaktiv analys.\n- Enkel installation och administration.\n\n## Arkitektur\nSystemet är uppdelat i fyra huvudkomponenter:\n\n- **Backend/Kommunikation**: Ansvarar för att hämta data från PLC:er via protokoll som Mitsubishi SLMP och Modbus TCP.\n- **Databas**: Lagrar all tidsseriedata, larmhistorik och systemkonfiguration.\n- **Frontend/SCADA-UI**: Ett webbaserat gränssnitt för att visualisera data, hantera larm och konfigurera systemet.\n- **AI/Felsökning**: Analyserar insamlad data för att identifiera avvikelser, förutse problem och assistera vid felsökning.\n\nSe `README.md` i respektive mapp för mer detaljerad information.
