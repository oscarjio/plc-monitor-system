# Databas\n\nDenna komponent hanterar all datalagring.\n\n## Datatyper\n- **Tidsseriedata**: Högupplöst data från PLC:er (t.ex. sensorvärden, motorstatus). En tidsseriedatabas som InfluxDB eller TimescaleDB är att föredra.\n- **Larm**: Aktiva och historiska larm, inklusive tidpunkt, prioritet och kvitteringsstatus.\n- **Händelselogg**: Viktiga systemhändelser och användarinteraktioner.\n- **Konfiguration**: Inställningar för PLC-anslutningar, taggar, vyer och användare.
